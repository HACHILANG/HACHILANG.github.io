Syntax Highlighting

HACHI does not have syntax highlighting in most editors, but in Mitred, you can freely have syntax highlighting.

~/.mitrc

Suppose you want to run some shellscript right before opening the editor, well put that in the file ~/.mitrc, and Mitred will run the content of that file.
